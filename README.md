# markertracker
tracks a marker model consisting of 4 UV-leds 

# dependencies 
Eigen, SFML, glew, image magick, assimp, opengl, OpenCV, sdformat, pcl, boost

# build
```
#!bash
cd path/to/markertracker
mkdir build
cd build
cmake ..
make 
```
# run
```
#!bash
cd path/to/markertracker/bin
./markertracker
```
